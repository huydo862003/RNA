# Surveys

- Ramalingam & Reps, "A Categorized Bibliography on Incremental Computation" (1993)
  - Comprehensive survey of the field up to 1993. Categorizes work into: value caching / memoization, static dependency analysis, dynamic dependency tracking, and finite differencing. Useful for understanding how the different approaches relate.
  - [ACM Digital Library](https://dl.acm.org/doi/10.1145/158511.158710)
